This project is for Software 2 C195. It is a JDBC that records customers
and appointments.

IntelliJ IDEA 2020.2.3 (Community Edition)
Build #IC-202.7660.26, built on October 6, 2020
Runtime version: 11.0.8+10-b944.34 amd64
JavaFX-SDK-11.0.2

To run the program, locate main in the model folder build and run
the application.

The additional report allows the user to select a customer by name
and retrieve a schedule of all their appointments.
